Fonctionnalités Clés — Code de Base\n7.1 Moteur de Pricing Dynamique\nLe prix d'une table évolue automatiquement selon 3 variables. Voici la fonction de calcul à placer dans src/lib/utils.ts :\n// src/lib/utils.ts\ninterface PricingParams {\nbasePrice: number        // Prix de base de la table\noccupancyRate: number    // Taux de remplissage 0–1 (ex: 0.7 = 70% plein)\ndaysUntilEvent: number   // Jours restants avant la soirée\neventNotoriety: number   // Coefficient notoriété DJ : 1.0 à 2.5\n}\nexport function calculateDynamicPrice({\nbasePrice, occupancyRate, daysUntilEvent, eventNotoriety\n}: PricingParams): number {\n// Coefficient occupation : monte fort quand >70% plein\nconst occupancyCoeff = occupancyRate < 0.5 ? 1.0\n: occupancyRate < 0.7 ? 1.15\n: occupancyRate < 0.85 ? 1.35\n: occupancyRate < 0.95 ? 1.65\n: 2.0\n// Coefficient délai : early bird = prix bas\nconst timeCoeff = daysUntilEvent > 14 ? 0.85\n: daysUntilEvent > 7  ? 1.0\n: daysUntilEvent > 3  ? 1.15\n: daysUntilEvent > 1  ? 1.35\n: 1.5  // Réservation le jour J\nconst finalPrice = basePrice * occupancyCoeff * timeCoeff * eventNotoriety\n// Arrondi au multiple de 25€ le plus proche\nreturn Math.ceil(finalPrice / 25) * 25\n}\n7.2 Système de Liens Promoteurs Trackés\nQuand un client arrive sur /reserve?promo=CODE, le code est capturé et stocké dans la session puis attaché à la réservation.\n// src/app/(public)/reserve/page.tsx\nexport default function ReservePage({\nsearchParams }: { searchParams: { promo?: string } }\n) {\nconst promoCode = searchParams.promo ?? null\n// Stocker le promo code dans un cookie 48h\n// Il sera lu au moment de créer la réservation\n}\n// src/lib/reservation.actions.ts — Au moment de créer la resa :\nconst promoCode = cookies().get('nighttable_promo')?.value\nif (promoCode) {\n// 1.
